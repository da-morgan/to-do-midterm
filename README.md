# Smart Todo List

- This project uses jQuery, Bootstrap,

## Getting Started


1. Install dependencies: `npm i`
2. Run the server: `npm run local`
3. Visit `http://localhost:8080/`
4. Add Todos!

## Dependencies

- Node 5.10.x or above
- NPM 3.8.x or above
- Bodyparser
- Express
- PSQL
